小猫可以分配工作，以获得 <a href="?file=003-资源大全/005-资源介绍">资源</a>
			，如
	<a href="?file=003-资源大全/03-矿物">
				矿物
	</a>
			，
	<a href="?file=003-资源大全/03-科学#科学">
				科学点
	</a>
			和
	<a href="?file=003-资源大全/03-文化#文化">
				文化
	</a>
			。
			分配作业的小猫越多，资源的收集速度就越快。
注意：研究
	<a href="?file=001-猫咪百科/04-工坊/01-升级#强化">
				强化
	</a>
			可以增加小猫所有工作的效率
