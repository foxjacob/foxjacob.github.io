# 难得素
  <p>
<strong>
难得素
</strong>
是一个晚期的游戏
<a href="?file=003-资源大全/005-资源介绍">
资源
</a>。
 </p>

### 制作


<ul>
  <li>
  在你研究
<a href="?file=001-猫咪百科/03-科学/01-科学#火箭学">
    火箭学
</a>
  和完成
<a href="?file=001-猫咪百科/07-空间/04-月球">
    探索月球任务之后
</a>
  ，你可以建立
<a href="?file=001-猫咪百科/07-空间/04-月球#月球前哨">
    月球前哨
</a>
  ，每轮生产0.35
<a href="#uranium">
    铀，
</a>
  每秒
  产生0.007个
<a href="#uranium">
    铀
</a>
  。
  </li>
  <ul>
<li>
    铀-1,750 / 秒 至难得素 0,035 / 秒
</li>
<li>
    这意味着500铀只能得到10个难得素。（（500铀/ 1,750s）* 0,035s = 10难得素））
</li>
  </ul>
</ul>
