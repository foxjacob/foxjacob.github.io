      </li>
      <li>
        <a href="?file=001-猫咪百科/01-建筑物/07-文化建筑#教堂">

      </li>
      <li>
            每个
          <a href="?file=001-猫咪百科/01-建筑物/07-文化建筑#寺庙">

            每0.2秒产生0.1个文化。
      </li>
      <ul>
        <li>
              在
          <a href="#Religion#Stained_Glass">
                彩色玻璃
          </a>

        </li>
        <ul>
          <li>
                购买
            <a href="#Religion#Transcendence">
                  超越
            </a>

          </li>
        </ul>
        <li>
              在
          <a href="#Religion#Basilica">
                大殿
          </a>

        </li>

          <li>

          </li>
        </ul>
      </ul>
    </ul>
    
### 用途
<ul>
      <li>
            文化
            在
          <a href="#workshop">
              作坊
          </a>
            把
        <a href="#parchment">
              羊皮纸
        </a>
            变成
        <a href="#manuscript">
              手稿
        </a>
            。
      </li>
      <li>
            建造
        <a href="?file=001-猫咪百科/01-建筑物/07-文化建筑#教堂">
              教堂
        </a>
            需要花费250文化。
      </li>
      <li>
            举办
        <a href="#festival">
              节日
        </a>
            花费5000文化。
      </li>
      <li>
            你积累了1500文化后，可以
            和娜迦
        <a href="?file=001-猫咪百科/05-贸易">
              贸易
        </a>
            ，
      </li>
    </ul>
